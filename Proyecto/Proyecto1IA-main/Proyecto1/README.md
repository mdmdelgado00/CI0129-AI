# User Manual: Run the brain_tumor_classification.ipynb File

## Prerequisites

Before starting, ensure you have the following installed:

1. [Visual Studio Code](https://code.visualstudio.com/download).
2. [Python](https://www.python.org/downloads/) (Python 3.7 or higher is recommended).

## Step 1: Download the Required Libraries

To run the program, you will need the following libraries: TensorFlow, NumPy, Matplotlib, Seaborn, and Scikit-Learn. If any of these are missing, follow these steps.

1. Open a terminal.
2. Run the following command: `pip install tensorflow numpy matplotlib seaborn scikit-learn`.

## Step 2: Install the Jupyter Extension in Visual Studio Code

1. Open **Visual Studio Code**.
2. Go to the **Extensions bar** by pressing `Ctrl+Shift+X`.
3. Search for the **Jupyter** extension in the search bar and click **Install**.

## Step 3: Select the Python Kernel

1. Open the **brain_tumor_classification.ipynb** file in Visual Studio Code.
2. At the top right of your notebook, you'll see an option to select the **Kernel** (the Python environment your notebook will use).
3. Click **Select Kernel** and choose the Python version you installed in the previous steps or an existing one.

## Step 4: Run the Program

1. Select the first cell of the notebook.
2. To run each cell, press `Shift + Enter`, or press the **Play** button at the top left corner of the cell.
3. It is recommended to run the cells in order.
